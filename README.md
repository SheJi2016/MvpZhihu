# MvpZhihu
MVP知乎日报重构

![主页](http://img.blog.csdn.net/20160711230108618)

## Features - 特性
  - 缓存功能
  - 典型的MVP模式
  - 不需要多余的权限
  - 没有广告
  - 没有推送消息，甚至没有任何的后台进程
  - 没有追踪模块，不收集您的任何隐私信息

## Dependency - 依赖
 - rxandroid:1.1.0
 - rxjava:1.1.3
 - retrofit:2.0.2
 - retrofit2:converter-gson:2.0.2
 - retrofit2:adapter-rxjava:2.0.2
 - glide:3.5.2
